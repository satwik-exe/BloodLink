
# BloodLink Backend

Node/Express + MongoDB backend for BloodLink (connecting donors and hospitals).

## Quick Start

```bash
cp .env.example .env
# edit .env
npm install
npm run dev
```

Swagger docs: `http://localhost:${PORT or 4000}/api/docs`

## Seed
```bash
npm run seed
```

## Roles
- donor
- hospital
- admin
